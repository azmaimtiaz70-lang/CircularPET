# EcoCompAI

## Vision

To develop an AI-assisted decision support platform for sustainable composite material selection and manufacturing optimization.

---

## Research Question

Can machine learning assist engineers in selecting sustainable composite materials and manufacturing parameters while balancing mechanical performance, cost, and environmental impact?

---

## Objectives

1. Build a literature-based composite material dataset.

2. Develop machine learning models to predict mechanical properties.

3. Recommend sustainable material-process combinations.

4. Visualize engineering trade-offs.

5. Develop an interactive prototype.

---

## Scope

Included:

- Polymer matrix composites

- Sustainable materials

- Machine learning

- Optimization

Excluded:

- Real factory deployment

- Industrial IoT

- Commercial software
